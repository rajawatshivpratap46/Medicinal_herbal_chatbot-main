🚀 Overview

The Medicinal Herbal Chatbot is designed to assist users in exploring natural remedies by answering queries related to herbs, their benefits, and usage.

It leverages AI + NLP + Generative AI (LLM APIs) to understand user queries and deliver context-aware, personalized responses.

🧠 Key Features

    🤖 AI-Powered Chatbot – Understands user queries using NLP techniques
    
    🧾 Herbal Knowledge System – Provides detailed medicinal information
    
    🗣️ Voice-to-Text Input – Enables speech-based interaction
    
    💬 Chat History – Stores previous conversations for better experience
    
    🎯 Smart Recommendations – Suggests related herbs and treatments
    
    ⚡ Real-Time Response – Fast and interactive chatbot system


🛠️ Tech Stack

    👨‍💻 Frontend
    
    HTML, CSS, JavaScript
    
    ⚙️ Backend
    
    Flask
    
    🧠 AI & NLP
    
    Python
      
    NLP Techniques (Text Processing, Intent Understanding)
        
    LLM API (Groq / OpenAI-like APIs)
    
    📊 Data Handling
    
    Mysql
      
    Pandas, NumPy

🧩 System Architecture
 
    User Input (Text / Voice)
            ↓
    Frontend (React UI)
            ↓
    Backend API (Flask)
            ↓
    NLP Processing (Python)
            ↓
    LLM API (Response Generation)
            ↓
    Processed Response → User Interface

⚙️ How It Works

    User enters a query (e.g., “Which herb is good for cold?”)
    
    Input is preprocessed (tokenization, cleaning, normalization)
    
    NLP pipeline identifies intent and extracts keywords
    
    Query is sent to LLM API for intelligent response generation
    
    Backend processes and formats the response
    
    Final answer is displayed to the user


📌 Sample Queries

  “Best herbs for immunity?”
  
  “Natural remedies for headache”
  
  “Uses of Tulsi”
  
  “Herbs for digestion problems”

🔧 Installation & Setup
  
  1️⃣ Clone Repository
    git clone https://github.com/your-username/medicinal-herbal-chatbot.git
    cd medicinal-herbal-chatbot
  
  2️⃣ Install Dependencies
  
    Frontend:
      
    cd client
    npm install
      
    Backend:
      
    cd server
    npm install
    
    Python (AI/NLP):
    
    pip install -r requirements.txt
  
  3️⃣ Setup Environment Variables
  
    Create a .env file in backend:
    
    API_KEY=your_llm_api_key
    MONGO_URI=your_mongodb_connection
  
  4️⃣ Run Application
  
    Backend:
    
    npm start
    
    Frontend:
    
    npm start
